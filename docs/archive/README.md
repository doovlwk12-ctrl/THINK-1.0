# أرشيف الوثائق

هذا المجلد يضم وثائق قديمة أو نقطة تحقق (checkpoints) وملخصات تحديث سابقة، محفوظة للمرجعية.  
لا تُحدَّث دورياً.

- **CHECKPOINT_*** — لقطات حالة المشروع سابقاً  
- **RESTORE_CHECKPOINT** — تعليمات استعادة من نقطة تحقق  
- **UPDATE_SUMMARY*** — ملخصات تحديثات قديمة  
- **BACKUP_INSTRUCTIONS** — تعليمات النسخ الاحتياطي  
- **DEBUG_PLAN**, **STATUS_REPORT** — خطط وتقارير سابقة  
- **FIREBASE_***, **FIRESTORE_DESIGN** — وثائق Firebase (تم إزالة Firebase من المشروع)
