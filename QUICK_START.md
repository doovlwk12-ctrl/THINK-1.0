# دليل البدء السريع - منصة فكرة

## خطوات التشغيل

### 1. تثبيت التبعيات
```bash
npm install
```

### 2. إعداد قاعدة البيانات
```bash
# إنشاء قاعدة البيانات
npx prisma generate
npx prisma db push

# إضافة بيانات تجريبية (اختياري)
npm run db:seed
```

### 3. تشغيل المشروع
```bash
npm run dev
```

### 4. فتح المتصفح
افتح [http://localhost:3000](http://localhost:3000)
